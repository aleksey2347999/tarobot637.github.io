# Tarot Web App for Telegram

Это веб‑приложение для раскладов Таро, предназначенное для работы внутри Telegram (Telegram Web App).  
Оно полностью автономно (один HTML-файл) и может быть размещено на любом статическом хостинге, например GitHub Pages.

## ✨ Возможности

- Полная база из 78 карт Таро с толкованиями.
- Три типа расклада: **отношения**, **работа**, **общий**.
- Выбор количества карт: **1, 3, 5**.
- Эффектная анимация «тасовки».
- Интеграция с Telegram Web App (кнопка закрытия, отправка данных).
- Блок с ценами подписки и номером для оплаты (по заданию).
- Адаптивный тёмный дизайн.

## 🚀 Как разместить на GitHub Pages

1. **Создайте репозиторий** на GitHub (например, `tarot-webapp`).
2. **Загрузите файл `index.html`** в корень репозитория.
3. **В настройках репозитория** (Settings → Pages) выберите ветку `main` и папку `/ (root)`. Нажмите Save.
4. Через пару минут ваш сайт будет доступен по адресу:  
   `https://<ваш-username>.github.io/tarot-webapp/`

## 🔗 Подключение к Telegram-боту

1. Скопируйте ссылку на ваше веб‑приложение (например, `https://username.github.io/tarot-webapp/`).
2. В коде вашего Telegram-бота (если он есть) укажите эту ссылку в параметре `WEB_APP_URL`.
3. При нажатии на кнопку с `WebAppInfo` бот откроет ваше приложение прямо в Telegram.

## 📱 Использование внутри Telegram

- Приложение автоматически разворачивается на весь экран.
- Кнопка **«ЗАКРЫТЬ»** возвращает пользователя в чат с ботом.
- При каждом раскладе в бота отправляются данные (тип расклада и список карт) – можно использовать для статистики.

## 💎 Информация о подписке

В приложении отображаются тарифы и номер для оплаты (согласно вашему заданию):
- 1 месяц – 120 ₽
- 6 месяцев – 450 ₽
- 12 месяцев – 600 ₽
- Номер: **+7 (953) 010-90-90**

Пользователь делает перевод и отправляет скриншот в бота – активация подписки происходит вручную администратором.
